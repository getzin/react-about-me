# React Portfolio

A personal portfolio website built with React, TypeScript and Vite.

## Features

- 🌙 Light / Dark Mode
- 👤 About Me
- 💻 Programming Skills
- 🌍 Languages
- 🎵 Favorite Albums
- 📬 Contact Section
- 📦 GitHub Projects (GitHub REST API)

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- Tailwind CSS
- GitHub REST API

## Getting Started

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
